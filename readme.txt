IMAGE LABELTOOL

	This tool is exploit for image label,before you use the image to train.

Requirements:lib

	opencv

Installation

1.Clone the image_labeltool
	git clone https://github.com/Robertchenh/iamge_labeltool
2.Build
	cd image_labeltool
	cmake
	make



Using

1.put your images into the 'iamge_datas'
2.open the tool      ./iamge_labeltool
3.do your work easy, your label will in the 'image_label'
